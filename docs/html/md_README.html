Skirtingos Strategijos

Strategija Nr1

Programa sukuria 2 papildomus to paties tipo (vector, list, deque) konteinerius -"islaike","skolininkai". Tada tikrina kiekviena "students" konteinerio elementa,jeigu elementas <= 5 priskyria ji "skolininkai", kitu atveju priskyria ji "islaike"

Strategija Nr2

Programa sukuria 1 papildoma to paties tipo konteineri "islaike". Tada tikrina pradinio konteinerio elementus, elementui bunant >= 5, priskyria ji "islaike" ir istrina ji is pradinio konteinerio

Strategija Nr3

Programa surikiuoja pradinio konteinerio elementus didejimo tvarka(pagal vidurki), tada suranda pirmo elemento >= 5 pozicija. Konteinerio dalys >= 5 yra priskyriama naujam konteineriui "islaike" naudojant copy(splice su std::list) funkcija, galu gale istrina is pradinio elemento nukopijotus elementus
